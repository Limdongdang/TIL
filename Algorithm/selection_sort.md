## 선택정렬

n개의 정수 데이터로 구성되어 있는 집합의 데이터를 오름차순으로 정렬한다.


1. 정수 데이터 배열 list를 입력한다.
2. 정렬되지 않은 데이터 집합에서 최소값을 찾는다.
3. 2단계 에서 찾은 데이터를 정렬된 list 다음 자리에 놓는다.
4. 모두 정렬될 때까지 수행한다.

```c
for (s=0; s< n-1;s++){
  min = s;
  for (j = s+1;j < n; j++){
    if (a[min] > a[j]){ min = j;}//최소값 찾기
  temp = a[s];
  a[s] = a[min];
  a[min] = temp;
```
